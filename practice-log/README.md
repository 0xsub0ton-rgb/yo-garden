# 🧘 Practice Log

Трекинг времени и качества практик. Помогает видеть прогресс в динамике и находить темы для постов в канале.

## Структура записи

Каждый месяц - отдельный файл: `2026-06.md`, `2026-07.md` и т.д.

### Формат одной записи

```
## 2026-06-13

- **Время:** 07:30 - 08:45 (75 мин)
- **Тип:** [асаны / пранаяма / медитация / общая практика]
- **Качество:** ⭐⭐⭐⭐ (1-5)
- **Заметки:** что далось легко, что нет, инсайты
- **Пост:** [да / нет / идея]
```

## Итоги месяца (в конце файла)

- Всего практик: N
- Суммарное время: X часов
- Средняя длительность: Y мин
- Топ-инсайт месяца: ...
